# Bus Booking System - Backend

Firebase Cloud Functions backend API.

## Setup

```bash
npm install
```

## Development

```bash
npm run serve  # Start local emulator
```

## Deploy

```bash
npm run build
npm run deploy
```

## Functions

- User management (roles, authentication)
- Booking transactions
- Loyalty points system
- Notifications
- Support tickets
- Refund processing
- Login history tracking
